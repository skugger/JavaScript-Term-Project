

	Drag & Drop game

	Slalom

	This is a variation on the game that you learn to code when you go through the W3 Schools Canvas tutorial.
	Rather than avoiding walls, use the arrow keys to avoid the oncoming objects.
	The farther to the right you play, the faster the game goes, and the faster your score rises!
